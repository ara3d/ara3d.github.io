Current Projects

We are developing the Plato programming language.

Plato is a cross platform programming language that is particularly well suited for 3D graphics programming, and can generate JavaScript, TypeScript, C#, and more.

Plato is designed to be a language that can be used for teaching programming, but that is also appropriate for commercial applications. It is both robust and efficient.

Past Projects
