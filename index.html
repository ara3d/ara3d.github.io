About Us

We are a Montreal based start-up specializing in 3D software development for AEC professionals founded by Christopher Diggins.

The construction industry has an enormous opportunity to steamline processes, reduce costs, and leverage data, through the application of modern software technology practices and good data management practices.

Our expertise in software engineering, computer science, and data modeling can bring significant savings to any organization trying to streamline their processes and make better usage of the data.

Ara 3D Studio

Ara 3D Studio is an interactive real-time 3D environment that puts computational design within everyone's reach.

It is inspired by parametric design tools like Grasshopper and Dynamo, but is designed to enable the generation and modification of extremely large amounts of data while maintaining interactive frame-rates.

Ara 3D Studio comes with an extensive and easy to use open-source geometry library. This means your tools and workflows can be executed outside of Ara 3D studio. An AI tool aids in the generation of new tools and analyses, even for people with no programming experience.

A public beta is planned for March 1st, 2025.

Open-Source Projects

We manage or have contributed to a large number of open-source libraries. Here are a few of them:

  • Bowerbird - a C# scripting environment for Revit
  • Revit Utils - a set of open-source libraries for Revit
  • Plato.Geometry - a C# library for 2D and 3D math and geometry
  • Plato - a cross platform programming language
  • BFAST - An efficient serialization format for storing arrays of binary data
  • VIM WebGL Viewer - A 3D WebGL viewer for AEC data
  • VIM File Format - A file format optimized for rendering and querying very large BIM models
